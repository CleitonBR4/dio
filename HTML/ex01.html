Tags aprendidas:

Tags HTML são elementos fundamentais para estruturar e formatar o conteúdo em uma página da web. Cada tag desempenha um papel específico na organização e estilização do texto e dos elementos visuais. Abaixo, vou explicar brevemente o propósito de algumas das tags mencionadas:

  1. <h1> até <h6>: Essas são as tags de cabeçalho, usadas para definir títulos e subtítulos em uma página. <h1> é o título mais importante e <h6> é o menos importante.
  2. <p>: A tag de parágrafo é usada para criar blocos de texto. Ela define uma quebra de linha antes e depois do conteúdo, separando-o em parágrafos.
  3. <mark>: A tag <mark> é usada para destacar um trecho de texto, geralmente com um fundo colorido, para chamar a atenção do leitor.
  4. <small>: Essa tag é usada para reduzir o tamanho do texto, indicando que seu conteúdo é menos importante, como notas de rodapé ou informações secundárias.
  5. <i>: A tag <i> é usada para aplicar ênfase ou estilo itálico ao texto, indicando que ele difere do texto circundante.
  6. <u>: A tag <u> é usada para sublinhar o texto, embora seu uso seja menos comum devido à associação com links não visitados.
  7. <strong>: Essa tag é usada para aplicar ênfase mais forte ou estilo negrito ao texto, indicando que é de grande importância no contexto.
  8. <ol>: A tag de lista ordenada <ol> é usada para criar listas numeradas, onde cada item é precedido por um número.
  9. <ul>: A tag de lista não ordenada <ul> é usada para criar listas com marcadores, onde cada item é precedido por um símbolo, como um ponto.
  10. <li>: A tag <li> é usada para definir cada item em uma lista, seja ordenada ou não ordenada.
  11. <a>: A tag de âncora <a> é usada para criar links para outras páginas da web ou recursos. Você pode definir o destino do link usando o atributo `href`.
  12. <hr>: A tag <hr> é usada para criar uma linha horizontal que divide o conteúdo da página. Pode ser usado para separar seções distintas.
  13. <sub>: A tag <sub> é usada para formatar o texto como subscrito, posicionando-o ligeiramente abaixo da linha de base.
  14. <sup>: A tag <sup> é usada para formatar o texto como sobrescrito, posicionando-o ligeiramente acima da linha de base.
  15. <blockquote>: A tag <blockquote> é usada para formatar citações longas ou bloco de texto que precisa ser destacado do restante do conteúdo.

tags são apenas algumas das muitas disponíveis em HTML. Ao combiná-las e usá-las adequadamente, você pode criar páginas da web bem estruturadas e visualmente atraentes. É importante lembrar que, para criar páginas modernas e responsivas, é recomendável combinar essas tags com CSS para estilização e, quando apropriado, JavaScript para interatividade.

Mozilla Developer Network (MDN): O MDN é uma fonte amplamente confiável para informações sobre tecnologias web. Eles têm uma documentação completa sobre HTML e suas tags. Visite a seção sobre HTML Basics para mais detalhes.

W3Schools: O W3Schools oferece tutoriais interativos e referências sobre HTML, CSS e muitas outras tecnologias web. A seção sobre HTML Introduction pode ser um bom ponto de partida.

HTML Living Standard: O HTML Living Standard é um documento técnico que define as especificações atuais do HTML. É mantido pelo Web Hypertext Application Technology Working Group (WHATWG) e é uma referência completa para todas as tags HTML.
